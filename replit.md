# HTML/CSS/JS Replit Extensions Starter

## Overview
A simple HTML/CSS/JS Extension Starter for Replit Extensions, demonstrating basic API calls for file operations.

## Project Structure
- `index.html` - Main extension UI
- `script.js` - JavaScript functionality
- `style.css` - Styling
- `background.html` / `background.js` - Background scripts for the extension
- `extension.json` - Extension configuration

## Running the Project
The project is served using `npx serve -l 5000` on port 5000.
